# Braderhud - Ecommerce

Tienda online de ropa urbana y streetwear. Proyecto de ecommerce con diseño moderno y responsive.

## Descripción

Sitio web de ecommerce para la marca Braderhud, especializada en poleras oversize con estilo urbano. El proyecto incluye página de inicio, catálogo de productos y vista detallada de productos.

## Características

- Diseño responsive (mobile, tablet y desktop)
- Catálogo de productos con tarjetas
- Vista detallada de producto con galería de imágenes
- Navegación entre páginas
- Footer con newsletter y enlaces
- Estilos modernos con Bootstrap 5

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.8
- Font Awesome

## Estructura del proyecto

```
ecommerce/
├── index.html              # Página principal
├── productos.html          # Catálogo de productos
├── producto1.html          # Detalle de producto
├── assets/
│   ├── css/
│   │   └── styles.css     # Estilos personalizados
│   ├── img/               # Imágenes del sitio
│   └── js/
│       └── app.js         # Scripts JavaScript
```

## Instalación

1. Clonar el repositorio
2. Abrir index.html en el navegador
3. No requiere instalación de dependencias

## Uso

Navegar por las páginas usando el menú:
- Inicio: página principal con productos destacados
- Productos: catálogo completo
- Detalle: hacer clic en cualquier producto para ver más información
